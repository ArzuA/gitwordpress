Set the Background Image per page....

== Description ==

Now you can set different background image on each page. Full Background Manager plugin provides you to set image to the body background. Just install the plugin, activate and go to the page in wordpress admin. You can see the meta box at last in the page in admin. Upload an image by clicking the Choose Image button. See you uploaded image's preview in the meta box. Publish the page and you'll get the image in the background in webpage. Want different background image at different page ? Just follow the same process in another page. Want to change background image ? Simply upload again. :)
For more products : http://www.store.perceptionsystem.com/wordpress-theme-mobile-app-promption-themes-red.html

== Installation ==

1. Upload 'Full Background Manager' to the wordpress admin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Upload the images from the Background Image meta box in page.

== Frequently Asked Questions ==

1. What is use of Full Background Manager plug-in? 
Using this plug-in, you can simply set an image to the Webpage background.

2. How can I use Full Background Manager plug-in? 
Full Background Manager plug-in is very easy to use. You just have to install your plug-in like others and upload the image in the Background Image metabox placed in the page in admin.

3. How can I display Background Image? 
No need to add any shortcode. Just upload the image and see the page at front side.
